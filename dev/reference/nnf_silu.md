# Applies the Sigmoid Linear Unit (SiLU) function, element-wise. See [`nn_silu()`](https://torch.mlverse.org/docs/dev/reference/nn_silu.md) for more information.

Applies the Sigmoid Linear Unit (SiLU) function, element-wise. See
[`nn_silu()`](https://torch.mlverse.org/docs/dev/reference/nn_silu.md)
for more information.

## Usage

``` r
nnf_silu(input, inplace = FALSE)
```

## Arguments

- input:

  (N,\*) tensor, where \* means, any number of additional dimensions

- inplace:

  can optionally do the operation in-place. Default: FALSE

[`nn_silu()`](https://torch.mlverse.org/docs/dev/reference/nn_silu.md).
