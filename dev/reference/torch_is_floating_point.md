# Is_floating_point

Is_floating_point

## Usage

``` r
torch_is_floating_point(self)
```

## Arguments

- self:

  (Tensor) the PyTorch tensor to test

## is_floating_point(input) -\> (bool)

Returns TRUE if the data type of `input` is a floating point data type
i.e., one of `torch_float64`, `torch.float32` and `torch.float16`.
