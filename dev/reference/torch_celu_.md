# Celu\_

Celu\_

## Usage

``` r
torch_celu_(self, alpha = 1L)
```

## Arguments

- self:

  the input tensor

- alpha:

  the alpha value for the CELU formulation. Default: 1.0

## celu\_(input, alpha=1.) -\> Tensor

In-place version of
[`torch_celu()`](https://torch.mlverse.org/docs/dev/reference/torch_celu.md).
