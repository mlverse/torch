# Gets and sets the default floating point dtype.

Gets and sets the default floating point dtype.

## Usage

``` r
torch_set_default_dtype(d)

torch_get_default_dtype()
```

## Arguments

- d:

  The default floating point dtype to set. Initially set to
  [`torch_float()`](https://torch.mlverse.org/docs/dev/reference/torch_dtype.md).
