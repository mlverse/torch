# Bitwise_xor

Bitwise_xor

## Usage

``` r
torch_bitwise_xor(self, other)
```

## Arguments

- self:

  NA the first input tensor

- other:

  NA the second input tensor

## bitwise_xor(input, other, out=NULL) -\> Tensor

Computes the bitwise XOR of `input` and `other`. The input tensor must
be of integral or Boolean types. For bool tensors, it computes the
logical XOR.
