# Bitwise_not

Bitwise_not

## Usage

``` r
torch_bitwise_not(self)
```

## Arguments

- self:

  (Tensor) the input tensor.

## bitwise_not(input, out=NULL) -\> Tensor

Computes the bitwise NOT of the given input tensor. The input tensor
must be of integral or Boolean types. For bool tensors, it computes the
logical NOT.
