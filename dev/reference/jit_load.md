# Loads a `script_function` or `script_module` previously saved with `jit_save`

Loads a `script_function` or `script_module` previously saved with
`jit_save`

## Usage

``` r
jit_load(path, ...)
```

## Arguments

- path:

  a path to a `script_function` or `script_module` serialized with
  [`jit_save()`](https://torch.mlverse.org/docs/dev/reference/jit_save.md).

- ...:

  currently unused.
