# Multiply

Multiply

## Usage

``` r
torch_multiply(self, other)
```

## Arguments

- self:

  (Tensor) the first multiplicand tensor

- other:

  (Tensor) the second multiplicand tensor

## multiply(input, other, \*, out=None)

Alias for
[`torch_mul()`](https://torch.mlverse.org/docs/dev/reference/torch_mul.md).
