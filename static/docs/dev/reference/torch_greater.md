# Greater

Greater

## Usage

``` r
torch_greater(self, other)
```

## Arguments

- self:

  (Tensor) the tensor to compare

- other:

  (Tensor or float) the tensor or value to compare

## greater(input, other, \*, out=None) -\> Tensor

Alias for
[`torch_gt()`](https://torch.mlverse.org/docs/dev/reference/torch_gt.md).
