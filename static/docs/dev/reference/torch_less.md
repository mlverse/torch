# Less

Less

## Usage

``` r
torch_less(self, other)
```

## Arguments

- self:

  (Tensor) the tensor to compare

- other:

  (Tensor or float) the tensor or value to compare

## less(input, other, \*, out=None) -\> Tensor

Alias for
[`torch_lt()`](https://torch.mlverse.org/docs/dev/reference/torch_lt.md).
