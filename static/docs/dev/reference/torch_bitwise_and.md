# Bitwise_and

Bitwise_and

## Usage

``` r
torch_bitwise_and(self, other)
```

## Arguments

- self:

  NA the first input tensor

- other:

  NA the second input tensor

## bitwise_and(input, other, out=NULL) -\> Tensor

Computes the bitwise AND of `input` and `other`. The input tensor must
be of integral or Boolean types. For bool tensors, it computes the
logical AND.
