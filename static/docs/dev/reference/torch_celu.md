# Celu

Celu

## Usage

``` r
torch_celu(self, alpha = 1L)
```

## Arguments

- self:

  the input tensor

- alpha:

  the alpha value for the CELU formulation. Default: 1.0

## celu(input, alpha=1.) -\> Tensor

See
[`nnf_celu()`](https://torch.mlverse.org/docs/dev/reference/nnf_celu.md)
for more info.
