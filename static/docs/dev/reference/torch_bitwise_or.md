# Bitwise_or

Bitwise_or

## Usage

``` r
torch_bitwise_or(self, other)
```

## Arguments

- self:

  NA the first input tensor

- other:

  NA the second input tensor

## bitwise_or(input, other, out=NULL) -\> Tensor

Computes the bitwise OR of `input` and `other`. The input tensor must be
of integral or Boolean types. For bool tensors, it computes the logical
OR.
